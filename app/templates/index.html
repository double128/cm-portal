Course Notices

Course Setup Guide:

Please upload your MyCampus CSV roster under Manage -> Users, then set a schedule under Manage -> Schedule, followed by a quota restriction under Manage -> Quota

Warning:

It is VERY important that you set a quota under Manage -> Quota to ensure students can't deploy more resources than they need in each lab! This quota represents the maximum number of VMs and other assets that can be deployed by each student. If this quota is too open, you may run out of resources.
